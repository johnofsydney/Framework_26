.container .bg-dark

  • The first element is .container .bg-dark
  • .container centres the content in the screen, with margins for larger screens.
  • .bg-dark sets the color of ther background and font.
  • This is a plain ul / li set. By default bullets are set to list-style:none
  • Default text aloign is centre (eg the h3 above), some elements such as ul and p have defaults set to left.

The container for this set of three is .space-between .bg-white

  • .space-between is a flex container, filling 100% of it's container.
  • It has no margin between itself and it's container and it has no internal padding to space it's leftmost and rightmost, topmost and bottom-most elements away from it's edges
  • .bg-white sets the background-color to solid white, and the font color to dark.
