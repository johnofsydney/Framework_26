The container for this set of three is .space-between .bg-white

  • .space-between is a flex container, filling 100% of it's container.
  • It has no margin between itself and it's container and it has no internal padding to space it's leftmost and rightmost, topmost and bottom-most elements away from it's edges
  • .bg-white sets the background-color to solid white, and the font color to dark.

This container is .space-between

The two images on the left are both 200 x 200px, the two on the right are both 400 x 200px

The elements within have no classes. The outside ones are plain old img and the inside two are img within div.

All img tags have a width: 100%; so they will entirely fill their container, whatever their actual size.

The only width information comes from the content of the element;

  • For the outside elements which are loose img tags, the width is the natura width of the image.
  • For the inside elements which are divs with img and p inside the width is determined by the other elements inside, in particular this text is what is stretching this colukn and making the smaller 200 x 200px cat appear larger than the 400 x 200px cat.

Both the outside images will distort as you change screen size
